Project Title
	Simple Shell project for Holberton Schools.

Project Description
	Simple Shell is simple UNIX command interpreter.
	It was created using C programming language.

How to use
	Run compiler: $ gcc *.c -o s_sh(DEYIS)
	Run simple shell: $ ./s_sh(DEYIS)
