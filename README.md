# Simple Shell Project

This project is a simple UNIX command interpreter created using the C programming language for Holberton School.

## How to Use

1. **Compile the Simple Shell:**
        ```
        $ gcc *.c -o simple_shell
        ```

2. **Run Simple Shell:**
        ```
        $ ./simple_shell
        ```
